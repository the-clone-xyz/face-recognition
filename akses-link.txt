# Akses Link Project Face Recognition

## Web App Browser
Local:
http://127.0.0.1:5001

Codespace:
Buka tab "Ports" di Codespace, cari port 5001, lalu klik "Open in Browser".

## API Web App
Status database:
http://127.0.0.1:5001/api/status

Daftar wajah:
http://127.0.0.1:5001/api/faces

## phpMyAdmin Asli
Local:
http://127.0.0.1:8080

Codespace:
Buka tab "Ports", cari port 8080, lalu klik "Open in Browser".

Login:
Username: facerec_user
Password: facerec_password
Database: facerecognition

## Database MySQL/MariaDB Lokal
Host: 127.0.0.1
Port: 3306
Database: facerecognition
User: facerec_user
Password: facerec_password

Command login terminal:
mysql -h127.0.0.1 -P3306 -ufacerec_user -pfacerec_password facerecognition

## Database Browser Internal
Alternatif selain phpMyAdmin:
http://127.0.0.1:5001/database
